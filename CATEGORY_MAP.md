# Category Map

documents: authoring and formatting for textual artifacts.
spreadsheets: tabular models, formulas, and workbook integrity.
presentations: slide generation and structure normalization.
data_serialization: json/csv/xml/yaml/toml data packaging.
diagrams: structured visual graph artifacts.
code_generation: executable script and source generation.
media_editing: image/video conversion and optimization.
validation_qa: artifact quality and compliance checks.
transformation_pipelines: cross-format and staged transformation flows.
prompt_contracts: structured prompt and instruction contracts.
skill_authoring: skill creation, metadata, and templates.
schema_design: schema generation and validation support.
artifact_routing: deterministic skill routing and dispatch.
metacognitive_support: session-state, workflow memory, and correction loops.
